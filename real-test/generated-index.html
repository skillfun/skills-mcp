常见问题

你可能有这些疑问

SkillHub 支持哪些 AI Agent 框架？▼
我们提供标准的 REST API 和多种语言 SDK（Python、JavaScript、Go、Rust），支持与 LangChain、AutoGPT、Claude Code、OpenAI Assistant 等主流框架无缝集成。
技能在运行时是否安全？▼
每个技能都在独立的 gVisor 沙箱中执行，拥有严格的资源配额和网络策略。所有调用均经过审计，支持细粒度的权限配置，企业版可私有化部署确保数据不落第三方。
我可以将自有工具发布为技能吗？▼
当然可以。使用我们的 Skill SDK，你可以将任何 API、CLI 工具或库包装为标准技能。提交后经过自动化审核即可上架，支持私有发布和公开上架两种模式。
免费版和付费版的区别是什么？▼
免费版提供每月 100 次技能调用和基础技能库访问，适合体验和原型验证。付费版解锁全部技能库、更高调用配额、自定义技能开发、优先支持等权益。
何时需要升级到企业版？▼
当你的团队需要不限调用次数、私有化部署、SSO 集成、SLA 保障或专属技术支持时，企业版是最佳选择。我们提供灵活的定制方案，欢迎联系销售团队咨询。
